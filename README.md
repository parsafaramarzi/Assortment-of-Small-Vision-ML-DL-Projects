These projects demonstrate how to implement advanced Deep Learning vision tasks using **Ultralytics** and **OpenCV**.

## 1. Real-Time Object Tracking (YOLOv8)
**File:** `YOLOv8 Car Traffic Detection.py`

This project implements object detection and tracking on a busy traffic video. It identifies vehicles and pedestrians, assigning them class labels and bounding boxes.

| Model | Input Source | Task |
| :--- | :--- | :--- |
| **`yolov8n.pt`** (Nano) | `Datasets/cartraffic03.mp4` | Multi-Object Tracking |

**Demo Output:**

<img src="output/yolo_demo_frame.png" width="100%" alt="Object Tracking Demo"/>

**Full Video:** [Download](output/yolo_detected.mp4)

> **Key Features:**
> * Tracks **80+ object classes** (COCO dataset).
> * Preserves original video aspect ratio during processing.
> * Lightweight "Nano" model for high-speed inference.

## 2. Real-Time Human Pose Estimation (YOLOv11)
**File:** `YOLO-Pose-PersonMoving.py`

A project focused on analyzing human movement. It detects people and estimates their body posture by tracking **17 specific keypoints** (skeleton) in real-time.

| Model | Input Source | Task |
| :--- | :--- | :--- |
| **`yolo11x-pose.pt`** (Extra Large) | `Datasets/personsport04.mp4` | Skeleton/Keypoint Detection |

**Demo Output:**

<img src="output/yolov11_pose_estimator_demo_frame.png" width="100%" alt="Pose Estimation Demo"/>

**Full Video:** [Download](output/yolov11_pose_estimator_detected.mp4)

> **Key Features:**
> * **Custom Visualization:** Draws a green skeleton and red keypoints manually for precise control.
> * **High Accuracy:** Uses the largest YOLOv11 model variant (`x`) for robust detection in dynamic sports footage.
> * Filters output to target only class `0` (Person).

## 3. Instance Segmentation & Crowd Analysis (YOLOv11)
**File:** `YOLOv11_Segmentation.py`

Unlike simple bounding boxes, this project performs **pixel-level segmentation**, drawing exact contours around objects. It is designed to handle dense environments like crowds.

| Model | Input Source | Task |
| :--- | :--- | :--- |
| **`yolo11x-seg.pt`** (Extra Large) | `Datasets/crowd06.mp4` | Instance Segmentation |

**Demo Output:**

<img src="output/yolov11_Segmentation_demo_frame.png" width="100%" alt="Segmentation Demo"/>

**Full Video:** [Download](output/yolov11_Segmentation.mp4)

> **Key Features:**
> * **Alpha Blending:** Uses semi-transparent masks (`alpha=0.5`) so the background remains visible.
> * **Enhanced Labels:** Implements custom text with **black outlines** to ensure readability against any color mask.
> * **Precise Boundaries:** Successfully separates overlapping individuals in a crowd.
